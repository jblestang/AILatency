# Simulateur de latence (pipeline 3 partitions)

Application egui pour simuler la latence d’un pipeline de traitement (16 Mbps, paquets 64–1500 octets) avec 3 partitions, files G/G/1 et budget 1 ms.

## Lancer l’app

```bash
cargo run
```

## Capture d’écran

Le fichier `screenshot.png` est une capture de l’interface. Pour la mettre à jour : lancer l’app puis faire une capture de la fenêtre (macOS : Cmd+Shift+4 puis Espace pour la fenêtre).
